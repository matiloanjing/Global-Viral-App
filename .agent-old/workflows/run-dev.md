# Run Development Mode

Run the GlobalViral Clipper from source code without building.

## Prerequisites

1. Python 3.8+ installed
2. FFmpeg in `./bin/` folder
3. Dependencies installed

## Steps

// turbo

1. Install dependencies (if not already):
   ```bash
   pip install customtkinter yt-dlp groq google-generativeai deep-translator edge-tts pillow
   ```

// turbo

2. Run the application:
   ```bash
   cd c:\Users\TO THE MOON\Downloads\BotYTS\OplusClip
   python main.py
   ```

## Notes
- API keys saved in `config.json`
- Temp files stored in `./temp/`
